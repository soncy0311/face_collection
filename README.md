### 실행 순서

- git clone

```bash
git clone https://github.com/thsckdduq/face_collection.git
```
<br>

- git clone한 face collection 폴더로 이동
```bash
cd face_collection
```
<br>

- requirements 다운로드

```bash
pip install -U pip

pip install -r requirements.txt
```
<br>

- dlib 패키지 다운로드
```bash
python -m pip install models\dlib-19.22.99-cp37-cp37m-win_amd64.whl
```
<br>

- data folder 만들기
